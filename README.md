# memoryStars

О приложении
Приложение показывает воспоминания в виде звёзд: у каждого воспоминания есть заголовок, дата, эмоция и теги.

Стек
- React + TypeScript + Vite
- @react-three/fiber + @react-three/drei (Three.js)
- Zustand, Tailwind CSS

Запуск
```powershell
pnpm install
pnpm dev
```
Открыть `http://localhost:5173`
